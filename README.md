# ☕ Coffee Shop Website

A modern, responsive coffee shop website built with HTML, CSS, and JavaScript. Features a clean design with smooth animations and an intuitive user interface.

## ✨ Features

- **Responsive Design** - Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Fixed Navigation Bar** - Sticky header with smooth navigation to different sections
- **Interactive Search** - Toggle-able search box with smooth animations
- **Hero Section** - Eye-catching landing page with call-to-action button
- **About Section** - Showcases the coffee shop's history and story
- **Product Gallery** - Grid-based product showcase displaying coffee offerings with prices
- **Customer Reviews** - Testimonial section with star ratings and customer feedback
- **Footer** - Comprehensive footer with support links, guides, and contact information
- **Smooth Animations** - Hover effects and transitions throughout the site

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with:
  - CSS Grid & Flexbox for layouts
  - CSS Variables for consistent theming
  - Media queries for responsiveness
  - Custom animations and transitions
- **JavaScript** - Interactive search functionality
- **Boxicons** - Icon library for UI elements

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project folder:
   ```bash
   cd Cofeeshop\ web\ site
   ```

3. Open `index.html` in your web browser

No build process or dependencies required!
